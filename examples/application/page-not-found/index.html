PageNotFound Examples

404 error page widget used by Application when no route matches. Also usable standalone.

1. Default Message

PageNotFound with no custom message — uses the built-in default text.

2. Custom Message

Pass a message param to override the description text.

3. Application Integration

In a real app, PageNotFound is shown automatically when navigating to an unknown route. Application.js uses it via the PageErrorCls config — you can override it with a custom error page.

class MyApp extends Application {
    constructor() {
        super({
            pathMap: { '/': 'pages/Home' },
            PageErrorCls: MyCustomErrorPage  // optional override
        })
    }
}

Try visiting #/nonexistent-route in the Simple SPA example to see it in action.
