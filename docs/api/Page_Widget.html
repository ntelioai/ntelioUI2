# new Widget(params)

Construct a new Widget. Handles template parsing, DOM node creation, registry registration, and optional auto-initialization.
Parameters:
Name Type Attributes Default Description
params Object Configuration object
template string <optional>
 HTML template string with {{key}} placeholders
templateName string <optional>
 ID of a template element inside #templates container
templateNode * <optional>
 Existing DOM node to wrap as the widget's root
node jQuery <optional>
 jQuery object to use as root, OR a plain DOM element to appendTo
replaceNode * <optional>
 DOM element this widget should replace in the document
widgetId number <optional>
 Explicit widget ID (auto-assigned if omitted)
register boolean <optional>
 true Set false to exclude from global registry
autoInit boolean <optional>
 true Set false to skip automatic init() + render() calls
onDestroy function <optional>
 Callback invoked after destroy completes
onRender function <optional>
 Callback invoked when render() is called
